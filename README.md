

# 📈 Zerodha Clone

A full-stack stock trading platform inspired by Zerodha, allowing users to manage portfolios, view holdings, place buy/sell orders, and track investments through a responsive dashboard.

## 🚀 Features
- User Authentication
- Buy & Sell Stocks
- Portfolio & Holdings Management
- Order History
- Responsive Dashboard
- REST API Integration

## 🛠 Tech Stack
- **Frontend:** React.js, React Router, Material UI, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Deployment:** Vercel, Render

## ⚙️ Installation

```bash
git clone <repository-url>
cd zerodha-clone
npm install
npm start
```
